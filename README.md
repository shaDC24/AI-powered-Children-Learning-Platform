# AI-powered-Children-Learning-Platform

# Project Title
Children's Learning Platform: a system that generates AI-based personalized learning contents for students.y & Research Ltd

# Brief Project Description 
Children’s Learning Platform is an AI-powered system developed by Y & Research Ltd. It studies each student’s learning pattern — how they learn, what they find difficult, and how fast they progress — to create personalized lessons and exercises.
The system automatically adjusts the content to match each student’s needs, helping them learn more effectively. Teachers and parents can also see the progress and get insights into each child’s strengths and weaknesses.
The goal is to make learning smart, personalized, and enjoyable for every student.

# Required Skills/Technologies

Programming Languages: Python, JavaScript (React, Node.js)
AI/ML Tools: PyTorch, Scikit-learn
Backend: FastAPI / Flask / Express.js
Database: PostgreSQL / MongoDB 
Frontend: React
